# Expense Tracker (HTML, CSS, JavaScript)

A simple Expense Tracker web application built using Vanilla JavaScript.  
This project allows users to manage their income and expenses and view their current balance.  
All data is stored in the browser using localStorage.

## Features

- Add income and expense transactions  
- Automatically calculate:
  - Total Income
  - Total Expenses
  - Current Balance  
- Transaction history with date and time  
- Delete individual transactions  
- Clear all transactions  
- Data persists using localStorage  
- Currency displayed in Indian Rupees (₹)  
- Simple and beginner-friendly UI  

## Tech Stack

- HTML  
- CSS  
- JavaScript (Vanilla JS)

## Local Storage

- Transactions are saved in browser localStorage  
- Data remains after page refresh  
- Clear All button removes all stored data  

## Validation Rules

- Transaction name cannot be empty  
- Amount must be a valid non-zero number  
- Negative values are automatically converted to positive
